# Code of Conduct

## Our Pledge

We pledge to make participation in this project respectful, useful, and welcoming for everyone.

## Expected Behavior

- Be kind and constructive.
- Assume good intent while still asking for evidence.
- Keep discussions public-safe: no secrets, customer data, or private project details.
- Focus criticism on ideas, workflows, docs, and code.

## Unacceptable Behavior

- Harassment, insults, or discriminatory language.
- Publishing private information without permission.
- Repeated off-topic or hostile comments.
- Sharing secrets, tokens, or private repository data in issues or pull requests.

## Enforcement

Maintainers may edit, hide, or remove comments and may restrict participation when behavior harms the project or community.

Report concerns through GitHub issues when public discussion is appropriate. Use GitHub private vulnerability reporting for security-sensitive concerns.
