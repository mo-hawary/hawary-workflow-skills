# Examples

Use these prompts after installing or pointing an agent at this skill.

## Full Audit

```text
Use dependency-security-auditor to scan this repository's dependency surfaces, separate complete coverage from weak evidence, and recommend safe remediation.
```

## Fast Pre-Push Check

```text
Use dependency-security-auditor to design a fast pre-push dependency CVE check that fails on high and critical vulnerabilities.
```

## CI Plan

```text
Use dependency-security-auditor to propose a CI dependency audit workflow for this repo, including required scanners, missing coverage, and fail thresholds.
```

## Native Scanner Only

```text
Use dependency-security-auditor to run available native ecosystem audits without OSV, then explain any incomplete coverage.
```
